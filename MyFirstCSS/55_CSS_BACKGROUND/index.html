CSS BASIC

  • כאמור, יש 3 דרכים להגדיר עיצוב: אחת, להשתמש בתכונה style בתוך התגית עצמה. דרך נוספת היא להשתמש בתגית style בתוך ה head והדרך האחרונה היא להשתמש בקובץ עיצוב חיצוני ולייבא אותו ל html.
  • ככל שהגדרת העיצוב קרובה יותר לתגית עצמה, כך ההגדרה "חזקה" יותר מבחינת ההיררכיה. תכונה > תגית > קובץ חיצוני.
  • למען השמירה על הסדר הטוב, נוחות השימוש ויעילות הקוד, הנטיה המוחלטת היא לעבוד עם קבצי CSS חיצוניים.
  • לטובת SEO ושיפור חווית המשתמש מומולץ מאד להשתמש בקבצי עיצוב מכווצים, בדרך כלל בעלי סיומת min.css.
  • הקריאה לקובץ עיצוב נעשת בתגית מסוג link, בתוך תגית ה head. ניתן לראות דוגמא בקובץ זה בשורה מספר 9.
  • כל הגדרה עיצובית ניתנת לסלקטור מסוים. הסלקטור יכול להיות לפי סוג התגית, ה- class או ה- id.

הגדרות עיצוב של טקסט

  • דוגמה לשורה שקיבלה הגדרות עיצוב מיוחדות לפי id
  • השורה הזו וזו שבאה אחריה קיבלו הגדרות עיצוב לפי אותו הקלאס
  • לשתי השורות יש קלאס font-control-2.

יחידות מידה של גודל

  • הגדרות עיצוב רבות צריכות לקבל ערך שכולל יחידות מידה של גודל.
  • יש יחידות מידה קבועות ויש יחידות מידה יחסיות. היום, עם הריבוי בגדלי המסכים וחששיבות העיצוב הרספונסיבי לחווית המשתמש ולדרוג ה- SEO, הולכת וגוברת הנטייה לשימוש ביחידות
  • יחידת המידה הקבוע הנפוצה ביותר היא 'px' והנפוצה אחריה היא 'pt' - מתקבלת בעורכי הטקסט של וורדפרס.
  • יחידות מידה יחסיות:
    • vw, view width - יחידת מידה שהיא מאית מרוחב המסך. ככל שהמסך קטן, גודל היחידה קטן איתו.
    • vh, view height - יחידת מידה שהיא מאית מגובה המסך. ככל שגובה המסך קטן, גודל היחידה קטן איתו.
    • rem - יחידת מידה יחסית לגודל הפונט שהוגדר לתגית ה html. אם לתגית ה html הוגדר גודל פונט של 16פקסל ולאחד מהדיבים הוגדר פונט בגודל 2rem אז הגודל של הפונט באותו הדיב יהיה 32פיקסל
