CSS COLORS, WIDTH AND HEIGHT

Colors

  • ישנם מספר אפשרויות להגדיר ערך של צבע ב css. למשל, את הצבע השחור ניתן להגדיר כך:
    • #000000 - Hex
    • #000 - Hex
    • rgb(0,0,0) - rgb
    • black
  • מגוון רחב של ערכי צבע ניתן לראות כאן
  • בערך הצבע ניתן להשתמש לתכונות רבות, ביניהן לצבע הפונט, צבע הרקע, צבע הגבול ועוד.
  • שימוש בצבע פונט ניתן לראות בקובץ העיצוב של דף זה, עבור הכותרות.
  • דוגמא לדיב עם צבע רקע אדום ופונט בצבע לבן

Width and Height

דיב עם גובה ורוחב ביחידות פיקסל

תרגילים

  • צרו 4 דיבים ולכל אחד הגדירו קלאס שונה.
  • לדיב הראשון קבעו אורך וגובה באחוזים ובחרו לו צבע רקע.
  • לדיב השני קבעו אורך וגובה ב vw ובחרו לו צבע רקע.
  • לדיב השלישי
    • הגדירו גודל פונט 4vw ותנו לו צבע רקע.
    • צרו בתוכו דיב נוסף והגדירו עבורו id = "inner-div"
    • לדיב הפנימי קבעו גובה ורוחב .5em והגדירו לו צבע רקע
    • בדקו את התוצאה שקיבלתם בדפדפן, תוך כדי שינוי גודל המסך.
    • הסבירו את ההתנהגות שקיבלתם
